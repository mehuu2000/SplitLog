# SplitLog

## 1. SplitLogとは

SplitLog は、macOS のメニューバーに常駐して使う作業ログアプリです。  
セッションごとに作業を区切り、`Split` 単位で経過時間とメモを記録できます。  
記録した内容はサマリーとしてコピーできるため、日報や作業メモへの転記がしやすい設計です。

## 2. 主な機能

- メニューバーからすぐに開ける軽量 UI
- セッションの追加・切り替え・削除
- `開始` / `停止` / `再開` / `Split` による時間記録
- Split ごとの名前編集とメモ記録
- サマリー生成とクリップボードコピー
- 表示設定（カラー/モノクロ、経過リング表示、リング周期）
- Split 配分モード切り替え（ラジオ/チェック）
- データ管理（セッション情報削除、Split情報削除、設定初期化、全データ初期化）

## 3. インストール方法（Slack配布ZIP）

1. Slack などで受け取った `SplitLog.zip` をダウンロードします。
2. `SplitLog.zip` をダブルクリックして解凍します。
3. 解凍してできた `SplitLog.app` を `アプリケーション` フォルダへ移動します。
4. `アプリケーション` フォルダ内の `SplitLog.app` を起動します。

注意:

- ZIP の中やダウンロードフォルダ上のアプリを直接使い続けず、必ず `アプリケーション` へ移動してから使ってください。
- 既存バージョンを置き換えるときは、先に SplitLog を終了してから上書きしてください。

## 4. 最初の使い方

1. `SplitLog.app` を起動します。
2. メニューバーのタイマーアイコンをクリックして本体 UI を開きます。
3. `開始` を押して計測を開始します。
4. 区切りたいタイミングで `Split` を押します。
5. 必要に応じて `停止` / `再開` を使います。
6. `サマリー` ボタンから内容を確認し、コピーして利用します。

## 5. 使い方ガイド

### セッション操作

- 右上の `+` でセッションを追加します。
- セッションチップで表示対象を切り替えます。
- ゴミ箱ボタンで現在セッションを削除できます。
- リセットボタンで現在セッションと Split を初期状態に戻せます。

### Split 操作

- `Split` ボタンで現在の区間を締めて次の Split に進みます。
- Split 行の左アイコンで配分対象を切り替えます。
- Split 名をタップすると名前を直接編集できます。
- ノートアイコンから Split メモを編集できます。

### サマリー

- サマリー画面では、メモ表示形式と時間表示形式を切り替えできます。
- コピーアイコンでサマリー全文をクリップボードへコピーできます。

### 設定

- 右上の歯車から設定画面を開けます。
- テーマ、表示、Split配分モード、サマリー表示形式を変更できます。
- ストレージ管理から、データ削除や初期化を実行できます。
- `SplitLogを終了` でアプリを終了できます。

## 6. データの保存先とバックアップ

主な保存先（通常）:

- セッションデータ  
  `~/Library/Containers/hamachi.SplitLog/Data/Library/Application Support/SplitLog/sessions.json`
- 設定データ  
  `~/Library/Containers/hamachi.SplitLog/Data/Library/Preferences/hamachi.SplitLog.plist`

バックアップ手順:

1. SplitLog を終了します。
2. 上記ファイルを任意の場所へコピーします。
3. 復元時は SplitLog を終了した状態で元の場所へ戻します。

## 7. よくある質問

### Q. 起動してもウィンドウが見当たりません
A. SplitLog はメニューバー常駐アプリです。画面右上のタイマーアイコンをクリックして開いてください。

### Q. 「開発元を確認できないため開けません」と表示されます
A. `SplitLog.app` を `control + クリック` して `開く` を選ぶか、`システム設定 > プライバシーとセキュリティ` から起動を許可してください。

### Q. 受け取った ZIP からそのまま起動してもいいですか？
A. 非推奨です。解凍後に `アプリケーション` フォルダへ移動してから起動してください。

### Q. それでも起動できない場合は？
A. ターミナルで次を実行して隔離属性を外したうえで、再度起動してください。

```bash
xattr -dr com.apple.quarantine /Applications/SplitLog.app
```

### Q. 別の Mac にデータを移せますか？
A. 可能です。`sessions.json` と設定ファイルをバックアップし、移行先の同じ場所へコピーしてください。
